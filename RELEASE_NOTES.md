# Release v1.0.0

发布日期: 2025-11-18

## 简短说明

首次发布：会议倒计时小工具（单文件静态页面）。包含界面与倒计时逻辑、签到与抽签功能，便于会议主持掌控环节时长。

主要文件：
- `countdown/index.html`

## 如何使用

1. 在本地打开：在浏览器中打开 `countdown/index.html` 即可使用（无需服务器）。
2. 或部署到静态站点：将 `countdown/` 目录上传到任意静态托管服务（例如 GitHub Pages、Netlify、Vercel 等）。
   - 若使用 GitHub Pages：可在仓库设置中启用 Pages，选择 `main` 分支并将目录设为 `/` 或者将 `countdown/` 内容移到 `docs/`。

## 打包与下载

- 发布包：`meeting-countdown-v1.0.0.zip`（已生成于仓库根目录）。
- GitHub Release：已在 GitHub 创建 Release，可通过以下链接访问：
- https://github.com/Liujian-unimind/Meeting-countdown/releases/tag/v1.0.0

## 其他说明

- 如需变更人员名单、环节时长等，可直接在页面中编辑或导入预设。欢迎提交 issue 或 PR 以改进功能。
